A simple helloworld app for docker

A simple nginx helloworld application that helps you learn docker image pulls. Runs on port :80

To pull this image:
```
docker pull crajapakshe/helloworld:latest
```

To run this image:
```
docker run -p 80:80/tcp "crajapakshe/helloworld:latest"
```
